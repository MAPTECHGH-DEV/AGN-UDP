# Installation


// Download the script
```
wget https://github.com/MAPTECHGH-DEV/AGN-UDP/raw/main/install_maptechudp.sh
```
// Edit script configuration 
```
nano install_maptechudp.sh
```
// Install the script
```
chmod +x install_maptechudp.sh; ./install_maptechudp.sh
```
